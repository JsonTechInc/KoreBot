# KoreBot
## 私有本地AI智能体 | 100% 本地运行 · 文件不出机 · 隐私安全

---

## 一、产品定位

KoreBot 是基于 OpenClaw 内核的本地私有 AI 智能体工具。它不依赖云端服务器、不上传用户文件、不收集隐私数据，全部运行在用户个人电脑上，通过浏览器即可使用。产品面向职场人士、学习者、投资者等人群，开箱即用，内置一系列贴合国内使用场景的自动化技能，是真正属于个人的本地 AI 助理。

---

## 二、核心理念

- 数据完全私有：所有文件、对话、任务均在用户本地电脑处理，绝不云端上传
- 轻量无负担：无需安装复杂客户端，双击启动，浏览器即用
- 国内友好体验：中文界面、固定工作目录、预置国内常用技能
- 模型成本自洽：内置免费 Token 体验，支持用户自行配置 Key，也可选择平台付费模型服务

---

## 三、产品形态与运行方式

1. 用户通过一键脚本（start.bat / start.sh）启动本地网关服务
2. 自动打开浏览器进入 KoreBot 管理界面
3. 系统自动划定本地工作目录，AI 仅可访问该目录下的文件
4. 所有 AI 调度、文件读取、技能执行均在本地完成
5. 联网仅用于模型接口调用与公开信息获取，用户资料不上云

---

## 四、核心能力

1. 全能对话交互
   以自然聊天方式完成提问、创作、总结、规划、分析等任务。

2. 本地文件智能处理
   直接读取本地文档、表格、文本，实现内容总结、要点提取、大纲生成。

3. 内置中文场景开箱即用技能
   - 本地文档精读与总结
   - 职场写作（周报、方案、邮件、PPT 提纲）
   - 学习与行业情报自动梳理
   - 股票公告 / 研报合规资讯辅助
   - 表格数据简单分析
   - 个人任务与计划管理

4. 灵活模型调用机制
   - 内置免费 Token
   - 支持用户自行配置第三方 API Key
   - 提供可选付费模型服务

5. 纯本地、绿色、无侵入
   - 不安装软件、不写注册表、不占用桌面
   - 不用云服务器、无容器、无私有云部署

## 九、GitHub 开源项目架构

### 9.1 架构整体概览

KoreBot 开源架构基于 OpenClaw 原生架构做轻量化封装，整体分为 4 层：

1. 内核层：复用 OpenClaw 核心内核
2. 适配层：本地网关、一键启动、配置管理
3. 应用层：中文 Web 界面、技能中心、文件处理
4. 生态层：开源文档、贡献指南、技能模板

---

### 9.2 各层详细设计

#### 9.2.1 内核层
- 复用 OpenClaw 稳定版内核
- 仅做中文适配与配置优化
- 不修改内核核心逻辑

#### 9.2.2 适配层
- 本地网关服务（默认端口：32451）
- 一键启动脚本（Windows / macOS / Linux）
- 本地配置自动生成与加密存储
- 工作目录自动创建

#### 9.2.3 应用层
- 中文 Web 界面：Vue3 + Webpack
- 四大核心页面：聊天、技能中心、模型设置、系统设置
- 预置 6 大高频中文技能
- 本地文件解析（docx / pdf / txt / csv / xlsx）

#### 9.2.4 生态层
- 完整开源文档
- Issue / PR 规范
- Skill 开发模板
- 社区协作机制
